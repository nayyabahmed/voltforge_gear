# Tools

Every tool the handbook uses, when it is first needed, and what to do if you
do not have it. You do NOT need everything on day one - tools are acquired
progressively, like everything else in this project (see
[the guiding principles](style-guides-principles/GUIDING-PRINCIPLES.md)).

## Maintenance Rules

1. Add a tool the moment a chapter first requires it, with the chapter number.
2. Always record a fallback for readers who do not own the tool.
3. Safety-relevant tools must link to their rules in [SAFETY.md](SAFETY.md).
